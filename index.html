HTML and CSS Cheat Sheet

Table Tags

Tag Name Description
table Table tags Defines the beginning and end of a table. All other table elements are nested inside this tag.
thead Table header Defines the header section of a table, typically used to group the heading rows.
tbody Table body Groups the body content of a table, where the main data rows are placed.
tfoot Table footer Defines the footer section of a table, typically used for summary information or totals.
tr Table row Represents a row in a table, containing either header cells or regular cells
th Table head Defines a header cell in a table. Text inside a table head tag is bold and centered by default.
td Table data Defines a standard cell in a table. It contains the actual data in a table row.

Table Attributes

Attributes Name Description
colspan Column span Specifies how many columns a cell should span across. It is still widely used in HTML tables to merge cells horizontally.
rowspan Row span Specifies how many rows a cell should span across. Like colspan, it is used to merge cells vertically.

HTML Tags

Tag Name Description
html html tag Defines the root of an HTML document. It wraps all other elements on the page.
title title tags Specifies the title of the web page, which appears in the browserâ€™s title bar or tab.
head head tag Contains meta-information about the HTML document, like the title, character set, and links to stylesheets.
div div tag A block-level container used to group elements for styling or layout purposes.
body body tag Contains the content of the HTML document, including text, images, links, tables, etc.
a Anchor tag Defines a hyperlink, which is used to link to other pages or sections within the same page.

HTML Attributes

Attributes Name Description
id Identifier Specifies a unique identifier for an element in an HTML document. Each id must be unique within the page. It is commonly used to target specific elements for styling with CSS or manipulation with JavaScript.
class Class identifier Specifies one or more class names for an element. Classes are used to apply styles to groups of elements or to target them in JavaScript. Multiple classes can be assigned to a single element, separated by spaces.
href Hypertext reference Specifies the URL for a link (used with the anchor tag). It tells the browser where the link should navigate to when clicked.
type Element type Specifies the type of an element, commonly used with input fields, scripts, and buttons. In forms, the type attribute defines what kind of input control will be displayed.

Pseudo Classes for Links and Buttons

These pseudo-classes help define the appearance of links and buttons during different stages of interaction with the user. The proper order of applying these classes is :link, :visited, :hover, and then :active.

Class Name Description
:link Link Class This pseudo-class selects links that have not yet been visited. It applies only to anchor elements with an href attribute.
:visited Visited Class This pseudo-class applies to links that the user has already visited. Browsers remember this state based on the browsing history.
:hover Hover Class This pseudo-class styles an element (such as a link or button) when the user hovers the mouse pointer over it. It is used for visual feedback on interactive elements.
:active Active Class This pseudo-class applies when a link or button is in the process of being clicked. It typically triggers when the user clicks and holds down the mouse button on the element.
